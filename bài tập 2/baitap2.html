API miễn phí dành cho việc ship hàng

Đây là cách dễ dàng nhất để tích hợp với nhiều hãng vận chuyển cho mục đích ship hàng của bạn.

Sử dụng API để tích hợp với các website thương mại điện tử hoặc phần mềm bán hàng của bạn.

Bạn có thể xây dựng một ứng dụng ship hàng của riêng mình

Restful APIs

Tất cả API đếu theo chuẩn Restful với ưu điểm là linh hoạt nhẹ,tương thích với nhiều hệ thống khác nhau.

Webhooks

Nhận update thông tin vận đơn qua webhook nhằm tối ưu hóa hiệu năng ứng dụng.

Tracking API

Nhận thông tin cập nhật mới nhất hoặc toàn bộ tiến trình di chuyển của vận đơn đày đủ nhất.
